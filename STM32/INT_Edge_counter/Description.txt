Прерывания (INT).
Внешние прерывание при нажатии кнопки на порту PA0.
При нажатии кнопки на порту PA0, по переднему фронту происходит
прерывание и обработчик этого прерывания производится счет количества
нажатий кнопки. Количество нажатий отображается на дисплее Nokia5110.
Прерывание без приоритета.
Для частичного устранения дребезга контактов кнопки используется задержа 200 мс.
Используется плата STM32 VL Discovery.
В программе используются символьные имена регистров, определенные
через define.
Использование PLL.
Не используется CMSIS Driver для SPI. 
Настройка только через регистры.
