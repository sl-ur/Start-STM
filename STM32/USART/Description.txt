USART - Универсальный синхронно/асинхроный прием/передача.
При нажатии кнопки происходит передача на компьютер, 
на терминальную программу, фразы "Hello,World!".
Так же синий светодиод выключается, включается зеленый светодиод.
Используется плата STM32 VL Discovery.
В программе используются символьные имена регистров, определенные
через define.
Для измения состояния битов портов используется регистр GPIOx_BSRR.
Использование PLL.
Не используется CMSIS Driver для USART. 
Настройка только через регистры.
